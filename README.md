# ToDoApp-demo

## 핵심기능 정리 

- 할 일 목록을 등록한다.
- 목록을 조회한다.
- 목록을 특정 id로 조회할수 있다.
- 완료한 일은 완료표시를 한다.
- 일 내용을 수정할 수 있다.
- 등록된 할 일을 삭제할 수 있다.
- 특정 id를 통해서 할 일을 삭제할 수 있다.
